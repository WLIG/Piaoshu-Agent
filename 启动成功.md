## 💡 使用技巧

### 保持服务器运行
- ⚠️ 不要关闭运行服务器的终端窗口
- 如果需要运行其他命令，打开新的PowerShell窗口

### 查看日志
- 服务器日志会实时显示在终端中
- 如果出现错误，查看终端输出

### 停止服务器
- 在服务器终端按 `Ctrl + C`
- 确认停止

### 重启服务器
```powershell
cd "C:\Users\Administrator\Desktop\Piaoshu  Agent  IP"
$env:DATABASE_URL = "file:./db/custom.db"
& "C:\Program Files\nodejs\npm.cmd" run dev
```

## 🐛 如果遇到问题

### 端口被占用
```powershell
# 查找占用端口的进程
netstat -ano | findstr :3000

# 结束进程
taskkill /PID <进程ID> /F
```

### 数据库错误
```powershell
# 重新推送数据库
$env:DATABASE_URL = "file:./db/custom.db"
& "C:\Program Files\nodejs\npx.cmd" prisma db push --force-reset
```
