## 🔧 常见错误解决方案

### 错误1：端口被占用
```
Error: listen EADDRINUSE: address already in use :::3000
```

**解决方案：**
```bash
# 查找占用进程
netstat -ano | findstr :3000

# 关闭进程（替换<PID>）
taskkill /PID <PID> /F

# 重新启动
npm run dev
```

### 错误2：模块未找到
```
Error: Cannot find module '...'
```

**解决方案：**
```bash
# 重新安装依赖
npm install

# 重新启动
npm run dev
```

### 错误3：Prisma错误
```
Error: @prisma/client did not initialize yet
```

**解决方案：**
```bash
# 生成Prisma Client
npx prisma generate

# 重新启动
npm run dev
```

### 错误4：编译错误
```
Error: Failed to compile
```

**解决方案：**
```bash
# 清理缓存
Remove-Item -Recurse -Force .next

# 重新安装
npm install

# 重新启动
npm run dev
```
