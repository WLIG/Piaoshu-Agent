## ⚡ 5 个命令启动系统

### 复制以下命令到终端，按顺序执行：

```bash
# 1️⃣ 安装依赖（2-5分钟）
npm install

# 2️⃣ 生成 Prisma Client（10秒）
npx prisma generate

# 3️⃣ 推送数据库（5秒）
npx prisma db push

# 4️⃣ 启动服务器（10-30秒）
npm run dev

# 5️⃣ 初始化数据（新终端，10秒）
curl -X POST http://localhost:3000/api/seed
```
