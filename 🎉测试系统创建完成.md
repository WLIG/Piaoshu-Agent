## 🔧 常见问题解决

### 如果服务器启动失败
```bash
# 清理缓存
Remove-Item -Recurse -Force .next

# 重新安装依赖
npm install

# 生成Prisma Client
npx prisma generate

# 重新启动
npm run dev
```

### 如果端口被占用
```bash
# 查找占用进程
netstat -ano | findstr :3000

# 关闭进程
taskkill /PID <进程ID> /F

# 重新启动
npm run dev
```
