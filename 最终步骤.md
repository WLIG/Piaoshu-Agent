## 🚀 最后一步：初始化数据

### 方法 1：使用 curl（推荐）

**打开新的终端窗口**，执行：

```bash
curl -X POST http://localhost:3000/api/seed
```

**预期输出：**
```json
{
  "success": true,
  "message": "Successfully seeded 6 articles",
  "data": {
    "created": 6,
    "skipped": 0
  }
}
```

---

### 方法 2：使用 PowerShell

```powershell
Invoke-RestMethod -Uri http://localhost:3000/api/seed -Method Post
```

---

### 方法 3：使用浏览器（最简单）

直接在浏览器中访问：
```
http://localhost:3000/api/seed
```

你会看到 JSON 响应显示在页面上。

## 🐛 常见问题

### 服务器没有启动
**检查：**
```bash
# 查看进程
netstat -ano | findstr :3000

# 如果端口被占用，使用其他端口
npm run dev -- -p 3001
# 然后访问 http://localhost:3001
```

### curl 命令不可用
**使用 PowerShell 或浏览器方法**（见上面）

### 页面空白
**检查：**
1. 打开浏览器控制台（F12）
2. 查看 Console 标签的错误
3. 查看 Network 标签的请求
4. 确认服务器正在运行

### 数据初始化失败
**重试：**
```bash
# 重新初始化
curl -X POST http://localhost:3000/api/seed
```
