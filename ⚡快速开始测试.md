# ⚡ 飘叔Agent - 快速开始测试

## 🚀 一键启动测试

只需运行一个命令，自动完成所有检查和测试：

```powershell
.\complete-test-flow.ps1
```

## 📝 这个脚本会做什么？

1. **检查配置** - 自动检查环境变量、数据库、依赖等
2. **自动修复** - 发现问题会提示并自动修复
3. **安装依赖** - 确保所有必要的包都已安装
4. **启动服务器** - 自动启动开发服务器
5. **运行测试** - 测试所有API和功能
6. **生成报告** - 显示详细的测试结果
7. **部署建议** - 根据测试结果提供下一步建议

## ✅ 测试内容

- 基础聊天API（/api/chat）
- NVIDIA模型API（GLM-4.7、Kimi 2.5等）
- 多媒体功能（图片、文档、视频分析）
- Skills系统
- 数据管理API
- 记忆系统
- 个性化系统
- 管理后台
- 前端页面
- API衔接和数据流

## 📊 测试结果

测试完成后会显示：
- ✅ 通过的测试
- ❌ 失败的测试
- 📈 通过率

**如果通过率 ≥ 80%，系统可以安全部署！**

## 🔧 如果遇到问题

脚本会自动尝试修复常见问题，包括：
- 创建缺失的环境变量文件
- 初始化数据库
- 安装缺失的依赖
- 生成Prisma Client
