## 🔧 快速命令

### 查看文章
```powershell
Invoke-RestMethod -Uri "http://localhost:3000/api/articles" -Method Get
```

### 搜索文章
```powershell
Invoke-RestMethod -Uri "http://localhost:3000/api/articles/search?q=飘叔" -Method Get
```

### 获取推荐
```powershell
Invoke-RestMethod -Uri "http://localhost:3000/api/recommendations?userId=anonymous&limit=5" -Method Get
```

### 发送消息
```powershell
$body = @{ message = "介绍一下飘叔"; userId = "anonymous" } | ConvertTo-Json
Invoke-RestMethod -Uri "http://localhost:3000/api/chat" -Method Post -Body $body -ContentType "application/json"
```

## 🐛 如果遇到问题

### 页面打不开
1. 确认服务器正在运行
2. 检查地址是否正确
3. 尝试刷新页面
4. 清除浏览器缓存

### AI不回答
1. 检查网络连接
2. 查看服务器终端日志
3. 尝试重新提问
4. 检查.env.local配置

### 推荐为空
- 这是正常的！
- 需要先浏览和点赞文章
- 系统会学习你的偏好
- 然后生成个性化推荐
