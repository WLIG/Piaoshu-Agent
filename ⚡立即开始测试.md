## 🚀 立即执行（3 步启动）

### 第 1 步: 打开新的命令提示符

1. 按 `Win + R`
2. 输入 `cmd`
3. 按回车

### 第 2 步: 进入项目目录

```cmd
cd "C:\Users\Administrator\Desktop\Piaoshu  Agent  IP"
```

### 第 3 步: 启动开发服务器

```cmd
npm run dev
```

**等待看到:**
```
  ▲ Next.js 15.x.x
  - Local:        http://localhost:3000
  
 ✓ Ready in Xs
```

## 🔧 如果启动失败

### 检查端口占用

```cmd
netstat -ano | findstr :3000
```

如果端口被占用，结束进程：
```cmd
taskkill /PID <PID> /F
```

### 重新安装依赖

```cmd
npm install
```

### 生成 Prisma 客户端

```cmd
npx prisma generate
```
