## 🎯 你需要做的

### 打开 PowerShell 或 CMD 终端

1. 按 `Win + R`
2. 输入 `powershell` 或 `cmd`
3. 按回车

### 进入项目目录

```bash
cd "C:\Users\Administrator\Desktop\Piaoshu  Agent  IP"
```

### 执行安装命令

```bash
npm install
```

**这个命令会：**
- 下载所有依赖包
- 创建 `node_modules` 文件夹
- 创建 `package-lock.json` 文件
- 大约需要 2-5 分钟

**等待看到：**
```
added XXX packages, and audited XXX packages in XXs

XXX packages are looking for funding
  run `npm fund` for details

found 0 vulnerabilities
```

## 🐛 如果遇到问题

### npm install 很慢
```bash
# 使用国内镜像
npm config set registry https://registry.npmmirror.com
npm install
```

### 权限错误
```bash
# 以管理员身份运行 PowerShell
# 右键点击 PowerShell -> 以管理员身份运行
```

### 网络错误
```bash
# 清理缓存后重试
npm cache clean --force
npm install
```
