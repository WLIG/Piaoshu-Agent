## ⚠️ 如果遇到问题

### 问题 1: 页面无法访问

**检查:**
- 服务器是否正在运行
- 端口 3000 是否被占用
- 防火墙是否阻止

**解决:**
```cmd
# 检查端口
netstat -ano | findstr :3000

# 重启服务器
Ctrl+C 停止
npm run dev 重启
```

### 问题 2: API 无响应

**检查:**
- .env.local 文件是否存在
- 环境变量是否正确
- API 密钥是否有效

**解决:**
1. 检查 .env.local 配置
2. 确认 API 密钥有效
3. 查看终端错误信息

### 问题 3: 聊天无回复

**可能原因:**
- API 密钥未配置
- 网络连接问题
- API 服务不可用

**解决:**
1. 检查浏览器控制台错误
2. 查看网络请求状态
3. 确认 API 密钥配置

## 📊 性能监控

### 关注指标

- **页面加载时间:** 应该 < 3秒
- **API 响应时间:** 应该 < 5秒
- **内存使用:** 保持稳定
- **CPU 使用:** 不应过高

### 优化建议

如果性能不佳：
1. 检查网络连接
2. 清理浏览器缓存
3. 重启开发服务器
4. 检查代码性能
