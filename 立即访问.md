## 🐛 常见问题

### Q: 端口 3000 被占用
```bash
# 使用其他端口
npm run dev -- -p 3001
# 然后访问 http://localhost:3001
```

### Q: 页面空白
1. 按 F12 打开控制台
2. 查看 Console 标签
3. 查看 Network 标签
4. 确认服务器正在运行

### Q: 数据初始化失败
1. 确认服务器正在运行
2. 重新访问 http://localhost:3000/api/seed
