## 🚀 立即部署 - 三步完成

### 步骤1: 推送代码到GitHub
```powershell
# 在PowerShell中执行
.\deploy-to-github.ps1
```

### 步骤2: 在Vercel中部署
1. 访问: https://vercel.com/wligs-projects
2. 点击 "New Project"
3. 选择 "WLIG/Piaoshu-Agent" 仓库
4. 点击 "Deploy"

### 步骤3: 配置环境变量（可选）
在Vercel项目设置中添加：
- `Z_AI_API_KEY` - 你的AI API密钥
- `OPENAI_API_KEY` - 你的OpenAI API密钥
- `NEXTAUTH_SECRET` - 生产环境密钥
