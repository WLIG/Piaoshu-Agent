## 🛠️ 手动启动步骤

### 步骤1: 检查环境
确保已安装：
- Node.js (版本16+)
- npm 或 yarn

### 步骤2: 安装依赖
```bash
npm install
```

### 步骤3: 配置数据库
```bash
npx prisma generate
npx prisma db push
```

### 步骤4: 启动服务器
```bash
npm run dev
```

### 步骤5: 访问应用
打开浏览器访问: **http://localhost:3000**

## 📊 功能亮点展示

### 1. 商业案例自动匹配
当用户询问平台经济时，Agent会自动引用：
- 淘宝连接买卖双方的案例
- 网络效应的商业逻辑
- 双边市场的价值创造

### 2. 智能类比生成
根据用户水平自动选择合适的类比：
- 初学者：生活类比 ("就像做菜一样...")
- 专家：技术类比 ("类似于分布式系统...")

### 3. 动态人格调整
根据对话内容自动调整表达风格：
- 商业话题：增强商业思维权重
- 技术话题：增强数据驱动权重
- 用户反馈：动态优化回复质量

### 4. 完整记忆管理
- 自动备份重要对话到数据库
- 支持记忆数据导出和导入
- 智能清理过期记忆数据

## 🔧 故障排除

### 常见问题
1. **端口占用**: 如果3000端口被占用，修改package.json中的端口
2. **依赖问题**: 删除node_modules文件夹，重新运行npm install
3. **数据库问题**: 重新运行prisma generate和db push
4. **编码问题**: 确保终端支持UTF-8编码

### 性能优化
- 首次启动可能较慢，请耐心等待
- 记忆系统使用内存存储，重启会清空数据
- 可以使用备份API保存重要记忆
