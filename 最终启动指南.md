# 🚀 飘叔Agent最终启动指南

## 🎯 快速启动步骤

### 1. 环境检查
```bash
# 检查Node.js版本
node --version
# 应该显示 v18+ 或更高版本

# 检查npm版本  
npm --version
```

### 2. 安装依赖
```bash
# 如果还没安装依赖
npm install

# 或者强制重新安装
npm install --force
```

### 3. 启动服务器
```bash
# 方法1：使用npx
npx next dev -p 3000

# 方法2：使用npm脚本
npm run dev

# 方法3：使用现有脚本
.\start-dev.ps1
```

### 4. 验证启动
- 等待30-60秒让服务器完全启动
- 打开浏览器访问：http://localhost:3000
- 看到飘叔Agent界面表示启动成功

## 🔧 故障排除

### 问题1：端口被占用
```bash
# Windows查看端口占用
netstat -ano | findstr :3000

# 终止占用进程
taskkill /PID <进程ID> /F

# 或者使用不同端口
npx next dev -p 3001
```

### 问题2：依赖问题
```bash
# 清理缓存
npm cache clean --force

# 删除node_modules重新安装
rmdir /s node_modules
npm install
```

### 问题3：编译错误
```bash
# 检查TypeScript错误
npx tsc --noEmit

# 如果有错误，可以跳过类型检查启动
npx next dev --turbo
```

### 问题4：权限问题
```bash
# 以管理员身份运行PowerShell
# 或者使用不同的包管理器
yarn dev
# 或
pnpm dev
```
