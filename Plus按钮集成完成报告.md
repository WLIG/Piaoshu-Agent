## 🎨 微信风格Plus按钮功能

### 界面布局
```
微信风格底部输入栏:
┌─────────────────────────────────────────┐
│  [➕] [输入你的问题...] [发送]            │
└─────────────────────────────────────────┘
```

### Plus功能菜单 (点击➕后展开)
```
┌─────────────────────────────────────────┐
│  [🎤]   [🖼️]   [📤]   [📄]              │
│  语音   图片   上传   文档              │
└─────────────────────────────────────────┘
```

### 功能特点
- **Plus按钮**: 圆形按钮，点击展开菜单
- **旋转动画**: 点击时45度旋转效果
- **4个功能**: 语音、图片、上传、文档
- **悬停效果**: 鼠标悬停时背景变化
- **响应式**: 桌面和移动端适配
