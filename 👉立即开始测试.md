# 👉 立即开始测试 - 简明指南

## 🎯 三步完成测试

### 第1步：启动服务器

**方式A（推荐）：** 双击运行
```
manual-start.bat
```

**方式B：** 在PowerShell中运行
```powershell
npm run dev
```

等待看到：`✓ Ready on http://localhost:3000`

---

### 第2步：快速测试

**打开新的PowerShell窗口**，运行：
```powershell
node quick-test.js
```

这会测试4个核心功能：
- ✅ 首页
- ✅ 聊天API
- ✅ NVIDIA API
- ✅ 简单聊天页面

**预期结果：** 至少3个通过 ✅

---

### 第3步：完整测试（可选）

如果快速测试通过，运行完整测试：
```powershell
node test-all-local.js
```

这会测试50+个功能点，包括：
- 所有聊天API
- NVIDIA多模型
- 多媒体功能
- Skills系统
- 数据管理
- 记忆系统
- 前端页面
- API衔接

**预期结果：** 通过率 ≥ 80% ✅

## ⚠️ 如果遇到问题

### 服务器启动失败
```powershell
# 检查端口占用
netstat -ano | findstr :3000

# 关闭占用进程
taskkill /PID <进程ID> /F

# 重新启动
npm run dev
```

### 测试失败
```powershell
# 运行错误检查
node check-and-fix-errors.js

# 查看详细文档
# 打开: 本地测试完整指南.md
```
