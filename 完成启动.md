## 🐛 常见问题

### Q1: 服务器启动失败
**A:** 检查端口是否被占用
```bash
# 查看端口占用
netstat -ano | findstr :3000

# 使用其他端口
npm run dev -- -p 3001
# 然后访问 http://localhost:3001
```

### Q2: 页面空白
**A:** 
1. 打开浏览器控制台（F12）
2. 查看 Console 标签的错误
3. 查看 Network 标签的请求
4. 确认服务器正在运行

### Q3: 数据初始化失败
**A:** 
1. 确认服务器正在运行
2. 确认访问的 URL 正确
3. 重新执行初始化命令

### Q4: curl 命令不可用
**A:** 
- Windows PowerShell 中使用 `Invoke-RestMethod`
- 或直接在浏览器访问 URL
