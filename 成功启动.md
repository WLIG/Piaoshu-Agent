### 初始化数据（3种方法任选其一）

#### 方法 1：浏览器（最简单）⭐⭐⭐⭐⭐
直接在浏览器打开：
```
http://localhost:3000/api/seed
```

#### 方法 2：curl
打开新终端：
```bash
curl -X POST http://localhost:3000/api/seed
```

#### 方法 3：PowerShell
```powershell
Invoke-RestMethod -Uri http://localhost:3000/api/seed -Method Post
```

**看到这个就成功了：**
```json
{
  "success": true,
  "message": "Successfully seeded 6 articles"
}
```
