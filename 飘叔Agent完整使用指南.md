### 🎤 **语音输入功能** - 语音转文字

#### 如何使用
1. 点击聊天框右侧的 **Plus按钮 (+)**
2. 选择 **"语音"** 选项
3. 点击麦克风按钮开始录音
4. 说完后再次点击停止录音
5. 系统自动识别并转换为文字
6. 自动发送消息获得AI回复

#### 使用技巧
- 🎙️ **清晰发音** - 说话清晰，语速适中
- 🔇 **安静环境** - 选择安静的环境录音
- ⏱️ **适当长度** - 建议每次录音1-30秒
- 🗣️ **中文优先** - 目前主要支持中文识别

#### 测试语音内容
```
🔹 短语测试：
"你好" / "谢谢" / "请问" / "帮我"

🔹 问题测试：
"请分析一下区块链技术的发展前景"
"Web4.0时代有什么特点？"
"如何提升用户体验？"

🔹 复杂问题：
"请详细分析一下这个商业模式的可行性，包括市场前景和风险评估"
```

#### 技术特性
- 🎯 **高准确率** - 85-98%的识别准确率
- ⚡ **快速响应** - 100-300ms处理时间
- 🔄 **备选结果** - 提供多个可能的识别结果
- 🚨 **错误处理** - 完善的错误提示和解决建议

### 🖼️ **图片分析功能** - AI视觉理解

#### 如何使用
1. 点击聊天框右侧的 **Plus按钮 (+)**
2. 选择 **"图片"** 选项
3. 上传图片文件或拖拽图片
4. 系统自动进行AI分析
5. 获得专业的图片解读和商业建议

#### 支持格式
- 📸 **图片格式** - JPG, PNG, GIF, WebP
- 📏 **文件大小** - 最大10MB
- 🖼️ **分辨率** - 支持各种分辨率

#### 分析维度
- 🔍 **内容识别** - 识别图片中的物体、场景、文字
- 💼 **商业价值** - 从商业角度分析图片的应用价值
- 🎯 **应用场景** - 推荐适合的使用场景和目标受众
- 💡 **优化建议** - 提供改进和优化的专业建议

### 📁 **上传功能** - 智能文档处理

#### 多媒体上传 (`/upload`)
1. 点击聊天框右侧的 **Plus按钮 (+)**
2. 选择 **"上传"** 选项
3. 进入多媒体上传页面
4. 拖拽或选择文件上传
5. 系统自动解析和处理

#### 文档上传 (`/upload/book`)
1. 点击聊天框右侧的 **Plus按钮 (+)**
2. 选择 **"文档"** 选项
3. 进入书籍文档上传页面
4. 设置书名和作者信息
5. 上传文档文件
6. 系统智能分割章节

#### 支持格式
```
🔹 文档格式：
- .docx (Word 2007+)
- .doc (Word 97-2003)
- .txt (纯文本，推荐UTF-8编码)
- .md (Markdown)

🔹 图片格式：
- .jpg, .jpeg, .png, .gif, .webp

🔹 其他格式：
- .pdf (部分支持)
```

#### 智能功能
- 🔤 **编码检测** - 自动检测文件编码，确保无乱码
- 📖 **章节分割** - 智能识别章节结构
- 📊 **内容解析** - 提取标题、内容、摘要
- 💾 **批量处理** - 支持多文件同时上传

#### 使用技巧
```
🔹 Word文档转换：
1. 打开Word文档
2. 文件 → 另存为
3. 选择"纯文本(*.txt)"
4. 编码选择"UTF-8"
5. 上传.txt文件

🔹 最佳实践：
- 使用UTF-8编码的.txt文件
- 文件大小控制在50MB以内
- 章节标题使用"第X章"格式
- 内容结构清晰，段落分明
```

## 🔧 故障排除

### 常见问题及解决方案

#### 🚨 服务器启动问题
```bash
# 问题：端口被占用
解决：netstat -ano | findstr :3000
     taskkill /PID <进程ID> /F

# 问题：依赖缺失
解决：npm install

# 问题：编译错误
解决：npm run build
```

#### 🎤 语音功能问题
```
问题：无法录音
解决：检查浏览器麦克风权限

问题：识别不准确
解决：在安静环境下清晰发音

问题：处理失败
解决：检查网络连接，重试录音
```

#### 🖼️ 图片分析问题
```
问题：上传失败
解决：检查图片格式和大小

问题：分析超时
解决：尝试压缩图片后重新上传

问题：结果不准确
解决：使用清晰、高质量的图片
```

#### 📁 上传功能问题
```
问题：文件乱码
解决：使用UTF-8编码保存文件

问题：解析失败
解决：检查文件格式，使用推荐格式

问题：章节识别错误
解决：使用标准的章节标题格式
```
