# 🚀 立即开始 - 飘叔 Agent

## ⚡ 3 步快速启动

### 第 1 步：打开终端

**Windows 用户：**
- 按 `Win + R`
- 输入 `powershell`
- 按回车

**或者：**
- 在项目文件夹空白处
- 按住 `Shift` + 右键
- 选择"在此处打开 PowerShell 窗口"

---

### 第 2 步：进入项目目录（如果需要）

```bash
cd "C:\Users\Administrator\Desktop\Piaoshu  Agent  IP"
```

---

### 第 3 步：复制粘贴以下所有命令

```bash
# 安装依赖
npm install

# 生成 Prisma Client
npx prisma generate

# 推送数据库
npx prisma db push

# 启动服务器
npm run dev
```

**然后在新终端执行：**
```bash
curl -X POST http://localhost:3000/api/seed
```
