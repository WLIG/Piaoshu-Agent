## 🔧 如果遇到问题

### 方案A：分步执行

```powershell
# 1. 进入目录
cd "C:\Users\Administrator\Desktop\Piaoshu  Agent  IP"

# 2. 安装依赖（5-10分钟）
npm install

# 3. 生成Prisma
npx prisma generate

# 4. 初始化数据库
npx prisma db push

# 5. 启动服务器
npm run dev
```

### 方案B：使用备用端口

如果3000端口被占用：
```powershell
npm run dev -- -p 3001
```
然后访问：`http://localhost:3001`

### 方案C：查看详细文档

打开 `手动启动指南.md` 查看完整说明
