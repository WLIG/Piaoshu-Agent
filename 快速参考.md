# ⚡ 飘叔 Agent 快速参考

## 🚀 5 个命令启动

```bash
# 1. 安装依赖（2-5分钟）
npm install

# 2. 生成 Prisma Client（10秒）
npx prisma generate

# 3. 推送数据库（5秒）
npx prisma db push

# 4. 启动服务器（10-30秒）
npm run dev

# 5. 初始化数据（新终端，10秒）
curl -X POST http://localhost:3000/api/seed
```

## 🐛 常见问题

### 端口被占用
```bash
npm run dev -- -p 3001
```

### curl 不可用（Windows）
```powershell
Invoke-RestMethod -Uri http://localhost:3000/api/seed -Method Post
```
或浏览器访问：http://localhost:3000/api/seed

### 重置数据库
```bash
npx prisma db push --force-reset
curl -X POST http://localhost:3000/api/seed
```
