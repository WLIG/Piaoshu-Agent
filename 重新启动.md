# 🔄 重新启动飘叔 Agent

## ⚠️ 服务器已停止

你需要重新启动服务器。

---

## ⚡ 快速重启（1步）

**在终端执行：**

```bash
npm run dev
```

**等待看到：**
```
✓ Ready in XXXms
○ Local:        http://localhost:3000
```

**⚠️ 保持终端运行！不要关闭！**

---

## 🌐 然后访问

```
http://localhost:3000
```

---

## ✅ 应该看到

- 飘叔Agent 标题
- 三个标签：推荐流、文章列表、对话
- 搜索框
- 文章列表

---

## 🐛 如果还是没有页面

### 检查 1：确认服务器正在运行
终端应该显示：
```
✓ Ready in XXXms
```

### 检查 2：测试 API
在浏览器访问：
```
http://localhost:3000/api/articles
```

应该看到 JSON 数据。

### 检查 3：硬刷新浏览器
按 `Ctrl + Shift + R`

### 检查 4：查看浏览器控制台
按 `F12`，查看 Console 标签是否有错误。

---

## 📚 详细排查

查看 **故障排除.md** 获取完整的故障排除指南。

---

## 🚀 现在就重启

**复制这个命令到终端：**
```bash
npm run dev
```

**等待 "Ready"，然后访问 http://localhost:3000**

---

**祝你成功！** 🎉
