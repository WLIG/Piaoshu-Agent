## 🐛 常见问题

### Q1: 端口 3000 被占用
```bash
# 查看占用进程
netstat -ano | findstr :3000

# 使用其他端口
npm run dev -- -p 3001
# 然后访问 http://localhost:3001
```

### Q2: 页面空白
1. 按 F12 打开浏览器控制台
2. 查看 Console 标签是否有错误
3. 硬刷新页面：Ctrl + Shift + R

### Q3: 编译错误
```bash
# 清理缓存
Remove-Item -Recurse -Force .next
# 重新启动
npm run dev
```
